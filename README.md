### Technologies
- Java 
- Spring Boot 
- Spring Data JPA
- PostgreSQL

### API Endpoints

#### Players
- `GET /api/players` - Lists all players
- `GET /api/players?name={name}` - Search by name
- `GET /api/players?team={team}` - Search by team
- `GET /api/players?position={position}` -  Search by position
- `GET /api/players?nation={nation}` - Search by nationality
- `GET /api/players/{id}` - Find player by ID
- `POST /api/players` - Adds a new player
- `PUT /api/players/{id}` - Updates a player
- `DELETE /api/players/{id}` - Removes a player
